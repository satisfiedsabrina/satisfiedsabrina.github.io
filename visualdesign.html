Artifact #7

Artist statement

I made this project with Adobe illustrator. The tools that I used were the brush tool to fill out my original picture. In this project I used 11 shades in my value scale and that was surprising because it didn't seem that much when I was using them. The hardest part about creating this project was making the picture look accurate because I didn't want it to look too cartoony and the easiest part was putting a different shade in each section of the photo since it was easier to see the light and darker parts.
